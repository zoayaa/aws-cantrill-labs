# AWS Cantrill Labs

This repository contains hands-on labs from the AWS Cantrill Associate course.  
All labs are built step-by-step on AWS, documented for learning and professional reference.

## Structure

- `vpc/` → Labs related to Virtual Private Cloud  
- `ec2/` → Labs related to EC2 instances, networking, and security groups  
- `iam/` → Labs related to IAM users, roles, and policies  

Each folder contains:
- README.md with objectives, architecture, steps, and learnings
- Screenshots or diagrams where applicable

## Goal

Build practical AWS skills while documenting every step for future reference and GitHub portfolio.
